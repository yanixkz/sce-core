Scientific examples (reproducible toy models)

SCE Core now highlights deterministic toy examples for CDS-oriented study. These are reproducible and inspectable, but they are not validated domain simulators or production science models.

Resource Stability

Resource-constrained population/regime selection under explicit constraints, including a walkthrough and a sensitivity grid experiment.

Walkthrough Sensitivity grid

Epidemic Regime

Toy epidemic regime stability under transmission, capacity, and intervention constraints, designed for deterministic cross-domain CDS checks.

Epidemic walkthrough

Hypothesis Research

Research-facing decision/evidence demo on the same engine, useful for inspectable ranking and next-step action structure.

Scenario docs

What you can run today

$ sce demo resource-stability
$ python examples/resource_stability_walkthrough.py
$ python examples/resource_stability_sensitivity.py
$ sce demo epidemic-regime
$ sce demo hypothesis
$ uvicorn sce.api:app --reload
# then open http://127.0.0.1:8000/docs and http://127.0.0.1:8000/ui

Alpha scope and limits

Early alpha

Interfaces and examples are evolving. Treat this as a compact research/product platform, not a finalized system.

Toy-model disclaimer

Scientific demos are deterministic, reproducible toy models meant for method study, not validated scientific simulators.

No broken docs assumptions

Swagger UI is a runtime FastAPI surface when running locally; GitHub Pages is static.
