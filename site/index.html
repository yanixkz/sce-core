SCE Core v0.1-alpha: computational framework for Constraint-Driven Stability

Mirror page: SCE Core is an early alpha computational framework for studying Constraint-Driven Stability: how constraints and dynamics select persistent structures from larger possibility spaces. The AI-agent decision API is an applied surface on the same framework.
